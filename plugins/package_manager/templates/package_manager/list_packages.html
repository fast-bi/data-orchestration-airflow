Package Installation Process:

  • The package will be added to the system requirements
  • Airflow components will be restarted to install the package
  • This process typically takes 5 minutes
  • Installation time may vary depending on package size and dependencies
  • During installation, some Airflow services may be temporarily unavailable
  • Use the Update button to change package versions without removing and re-adding

Please ensure critical workflows are not running during package installation.
